# P1 基础规则配置执行说明

## 1. 目标
完成启元农链第一阶段基础配置中心的数据库落库、菜单授权和本地联调。

## 2. 你需要执行的操作

### 2.1 安装并启用 JDK 17
当前工程编译目标为 Java 17，未满足会报错：无效的目标发行版: 17。

建议操作：
1. 安装 JDK 17。
2. 设置环境变量 JAVA_HOME 指向 JDK 17。
3. 将 %JAVA_HOME%/bin 放到 PATH 前面。
4. 打开新终端执行：
   - java -version
   - mvn -version
确认均显示 17。

### 2.2 执行数据库脚本
在若依数据库执行：
- sql/agri_base_rule.sql

该脚本会：
- 创建表 agri_base_rule
- 新增菜单目录“启元农链”和子菜单“基础规则配置”
- 新增按钮权限
- 初始化六类基础规则数据

### 2.3 菜单授权
给测试角色分配以下权限：
- agri:baseRule:list
- agri:baseRule:query
- agri:baseRule:add
- agri:baseRule:edit
- agri:baseRule:remove
- agri:baseRule:export

### 2.4 启动联调
后端：
- 在 RuoYi-Vue-master 目录执行 mvn -pl ruoyi-admin -am -DskipTests compile
- 启动 ruoyi-admin

前端：
- 在 ruoyi-ui 目录执行 npm install
- 执行 npm run dev

访问路径：
- 启元农链 / 基础规则配置

## 3. 验收清单
- 页面可打开并可分页查询。
- 可新增/编辑/删除规则。
- 规则类型可覆盖六类配置。
- 导出功能可正常下载。
